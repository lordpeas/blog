具名和匿名函数，是否有函数名

立即执行函数表达式
``` javascript
var a = 2;
(function foo(global) { 
  var a = 3;
  console.log( a ); // 3
})(window); 
 console.log( a ); // 2
 // 由于函数被包含在一对 ( ) 括号内部,因此成为了一个表达式,通过在末尾加上另外一个 ( ) 可以立即执行这个函数
 // 从函数外部作用域传 window 进去，函数内部看起来更加清晰
```
